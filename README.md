# Odin Recipes

A simple recipe website built as part of The Odin Project Foundations curriculum.

## Overview

The goal of this project was to practice the fundamentals of web development by creating a multi-page website using only HTML. The site contains a homepage that links to several recipe pages, each with ingredients, instructions, and images.

## Technologies Used

- HTML5
- Git
- GitHub

## Skills Practiced

- Semantic HTML
- Working with links and images
- Creating and organizing multiple webpages
- File structure and project organization
- Version control with Git

## Why I Built This

As I work through The Odin Project, I'm documenting my progress by pushing each completed project to GitHub. While the project itself is intentionally simple, it helped reinforce the importance of strong fundamentals before moving on to CSS, JavaScript, and modern frameworks.

## Future Improvements

- Add styling with CSS
- Improve accessibility
- Make the site responsive
- Expand the collection of recipes
